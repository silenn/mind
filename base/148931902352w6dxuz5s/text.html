printf


Рекомендации CERT: не использовать printf с одним аргументом и символом завершения строки. Вместо этого использовать puts.

Например, вместо:


printf("Hello world!\n\r");


Использовать:


puts("Hello world!");


Если необходимо вывести текст без перевода на новую строку, то рекомендуется использовать printf с 2 аргументами:


printf("%s", "Hello world!");
