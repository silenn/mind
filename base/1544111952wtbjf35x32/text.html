Если ПК подключён к интернету, то проще всего активировать Windows через какой-нибудь любительский KMS-сервер. Активация осуществляется по стандартному общему GVLK-ключу. Запускаем командную строку от имени Администратора, далее копируем и вставляем команды (Enter после каждой):

  1. SLMgr.vbs /IPK XXXXX-XXXXX-XXXXX-XXXXX-XXXXX
  2. SLMgr.vbs /SKMS kms.digiboy.ir
  3. SLMgr.vbs /ATO

***Наиболее популярные ключи

  1. Windows 7: Professional - FJ82H-XT6CR-J8D7P-XQJJ2-GPDD4
  2. Windows 7: Enterprise - 33PXH-7Y6KF-2VJC9-XBBR8-HVTHH
  3. Windows 10: Professional - W269N-WFGWX-YVC9B-4J6C9-T83GX
  4. Windows 10: Enterprise - NPPR9-FWDCX-D2C8J-H872K-2YT43  
    Полный список ключей доступен в библиотеке Microsoft TechNet, раздел "Ключи установки клиента KMS". (https://docs.microsoft.com/en-us/windows-server/get-started/kmsclientkeys)
