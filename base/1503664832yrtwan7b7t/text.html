Образование сложных существительных:


die Geburt (рождение) + der Tag = der Geburtstag (день рождения);

Соединительная <s> в середине для лучшего произношения. Артик берется от главного слова.

der Geburtstag + das Geschenk = das Geburtstagsgeschenk (подарок ко дню рождения);

das Geburtstagsgeschenk + das Papier = das Geburtstagsgeschenkpapier (упаковочная бумага для подарков ко дню рождения).


Существительные стоят исключительно после прилагательного или числительного:

Rote Johannisbeere (смородина красная)

Sibirische Pelmeni (пельмени сибирские)

Vielen Dank (спасибо большое)
