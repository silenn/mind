Macroses:





For Vanilla:

Кастуем на маусовер. Если маусовера нет, то на таргет, если и таргета нет, то на себя...

/run c=CastSpellByName s="Renew(Rank 9)" if UnitExists("mouseover") then TargetUnit("mouseover"); c(s); TargetLastTarget() else if UnitIsPlayer("target") then c(s) else c(s, 1) end end


Кастуем на маусовер. Если его нет, то - на таргет...

/run c=CastSpellByName s="Renew(Rank 1)" if UnitExists("mouseover") then TargetUnit("mouseover"); c(s); TargetLastTarget() else c(s) end
