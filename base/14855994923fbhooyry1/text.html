1. const с плавающей точкой всегда подразумевает тип double.

Поэтому объявлять, например, const float pi = 3.14; некорректно.

Правильно: const float pi = 3.14f; - это приводит число к типу float до записи в память и позволяет избежать warning'ов.

2. Символьная константа: 'A', '\101' (восьмеричное представление) или '\x41' (шестнадцатеричное представление).
