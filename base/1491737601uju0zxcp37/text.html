Тач хоть и работает из коробки, но настройкам практически не поддается. Вертикальный скрол мыши за 3 полных прохода пальцем по тачу... Надоело.


Посмотреть список дивайсов в/в:

$xinput list


Выключить или включить устройство соответственно:

$xinput --disable xx

$xinput --enable xx

Вместо хх указываеться id устройства.


Исправляется следующим образом:

cd /etc/modprobe.d/

sudo touch ./elantech-touchpad.conf

sudo echo "options psmouse proto=imps" > ./options.conf

sudo modprobe -r psmouse

sudo modprobe psmouse


После этого можно проверить, сделав run:

gpointing-device-settings


В настройках вместо Mouse появился ETPS/2 Elantech Touchpad

Даже с дефолтными параметрами тач стал шустрей.
