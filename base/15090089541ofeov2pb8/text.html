Бывает так что при открытии файла с русским языком Sublime показывает иероглифы вместо русских букв.

Исправить это можно двумя способами:

1. Reopen with Encoding

В верхнем меню меню идем в пункт File → Reopen with Encoding → выбираем нужную кодировку. В моем случае помогает Cyrillic (Windows 1251).

2. Settings

Идем в Preferences → Settings–Default и меняем строку:

"fallback_encoding": "Western (Windows 1252)"

на

"fallback_encoding": "Western (Windows 1251)"

или на

fallback_encoding": "Cyrillic (Windows 1251)”
